# Resources Folder

This folder contains assets for electron-builder packaging.

## Icon

To add a custom icon:

1. Create or download a 256x256 PNG icon
2. Convert to .ico format using online tools:
   - https://convertio.co/png-ico/
   - https://www.icoconverter.com/
3. Save as `resources/icon.ico`

## Default Behavior

If no icon is provided, electron-builder will use the default Electron icon.

## Icon Requirements

- Format: .ico
- Size: 256x256 pixels (recommended)
- Transparent background (optional)

## Creating an Icon

You can create a simple icon using:
- GIMP (free)
- Photoshop
- Online icon generators
- AI image generators (DALL-E, Midjourney)

## Noir Style Icon Suggestions

For a Noir/Savecord themed icon:
- Black background
- Neon cyan/blue accents (#3CAEA3)
- Minimalist design
- Discord-inspired but unique
- Shield or lock symbol (for security theme)
