## Базовая настройка

### Первый запуск

```bash
# 1. Установка зависимостей
npm install

# 2. Сборка проекта
npm run build

# 3. Инжект в Discord
node dist/injector.js

# 4. Перезапуск Discord
# Закройте Discord полностью (Ctrl+Q) и запустите снова
```

### Проверка установки

После запуска Discord вы должны увидеть:
- Quick Bar в верхней части окна (⚡☢️🔄👻📻)
- Сообщение в консоли: `[Savecord] Initialized`

## Оптимизация производительности

### Сценарий 1: Игра + Discord

**Проблема:** FPS падает когда Discord открыт

**Решение:**

1. Нажмите **📻 Radio Mode** в Quick Bar
2. UI Discord полностью замораживается
3. Голосовой чат продолжает работать
4. FPS возвращается к норме

**Результат:** +15-20 FPS в играх

### Сценарий 2: Медленный компьютер

**Проблема:** Discord тормозит на Pentium G850

**Решение:**

1. Нажмите **⚡ Optimize** в Quick Bar
2. Включается Nuclear CSS (нет анимаций)
3. Плагины обновляются реже (10s вместо 1s)
4. Отключаются тени и размытия

**Результат:** -50% нагрузка на CPU

### Сценарий 3: Экономия трафика

**Проблема:** Лимитированный интернет

**Решение:**

1. Откройте настройки Media Killswitch
2. Включите блокировку изображений/видео
3. Медиа будет показываться как placeholder

**Результат:** -80% трафика

## Приватность и безопасность

### Сценарий 1: Защита от RAT/стилеров

**Проблема:** Подозрение на заражение компьютера

**Решение:**

1. Нажмите **Ctrl + Alt + Shift + K** (Panic Button)
2. Все данные Discord очищаются
3. Автоматическая перезагрузка
4. Токены защищены от кражи

**Результат:** Экстренная защита токена

### Сценарий 2: Анонимность через Tor

**Проблема:** Нужна максимальная анонимность

**Решение:**

1. Установите Tor Browser
2. Запустите Tor (автоматически запустится SOCKS5 на 9050)
3. Отредактируйте `config/privacy.json`:

```json
{
  "useProxy": true,
  "proxyAddress": "socks5://127.0.0.1:9050"
}
```

4. Перезапустите Discord

**Результат:** Весь трафик через Tor

### Сценарий 3: Удаление EXIF из фото

**Проблема:** Не хотите раскрывать GPS координаты

**Решение:**

1. Onion Privacy автоматически включен
2. При загрузке фото EXIF удаляется
3. GPS, модель камеры, дата - всё стирается

**Результат:** Анонимные фотографии

### Сценарий 4: Проверка файлов на вирусы

**Проблема:** Получили подозрительный файл

**Решение:**

1. Security Shield автоматически сканирует `.exe`, `.dll`, `.zip`
2. Если обнаружен вирус - красная рамка вокруг сообщения
3. Тултип показывает: "⚠️ MALWARE DETECTED: 45/70"

**Результат:** Защита от вредоносных файлов

## Сетевые функции

### Сценарий 1: Discord заблокирован провайдером

**Проблема:** Не работает голосовой чат / видео

**Решение:**

1. Zapret автоматически запускается при старте
2. Пробует 3 стратегии DPI bypass
3. Нажмите **🔄 Refresh Zapret** для смены стратегии

**Результат:** Обход блокировок DPI

### Сценарий 2: YouTube не грузится в Discord

**Проблема:** Превью YouTube не показываются

**Решение:**

1. Zapret обходит блокировки YouTube
2. Если не помогло - нажмите **🔄 Refresh Zapret**
3. Попробуйте другую стратегию

**Результат:** YouTube работает

### Сценарий 3: Автообновление Zapret

**Проблема:** Старая версия Zapret не работает

**Решение:**

1. UpdateService автоматически проверяет GitHub
2. Скачивает новую версию `winws.exe`
3. Обновляет без перезапуска Discord

**Результат:** Всегда актуальная версия

## Кастомизация

### Сценарий 1: Кастомная тема

**Проблема:** Хочу свою цветовую схему

**Решение:**

1. Откройте Quick CSS плагин
2. Вставьте свой CSS:

```css
:root {
  --background-primary: #1a1a2e;
  --background-secondary: #16213e;
  --accent: #0f3460;
}
```

3. Сохраните

**Результат:** Своя тема

### Сценарий 2: Отключение телеметрии

**Проблема:** Discord собирает данные

**Решение:**

1. Onion Privacy автоматически блокирует:
   - `/science`
   - `/analytics`
   - `/metrics`
   - `sentry.io`
   - `google-analytics.com`

**Результат:** Нет телеметрии

### Сценарий 3: Просмотр удалённых сообщений

**Проблема:** Кто-то удалил важное сообщение

**Решение:**

1. Нажмите **🗑️** в тулбаре
2. Откроется Ghost Archive
3. Показывает последние 50 удалённых сообщений

**Результат:** История удалений

### Сценарий 4: Режим невидимки

**Проблема:** Не хочу показывать "печатает..."

**Решение:**

1. Нажмите **👻 Ghost Mode** в Quick Bar
2. Включается Typing Stealth
3. Никто не видит когда вы печатаете

**Результат:** Полная приватность

## Советы и трюки

### Совет 1: Экономия RAM

Отключите неиспользуемые плагины:

```typescript
// В src/preload/index.ts
deferred: ["message-logger"] // Закомментируйте ненужные
```

### Совет 2: Ускорение загрузки

Переместите плагины в `critical` для приоритетной загрузки:

```typescript
critical: ["your-plugin"]
```

### Совет 3: Отладка

Откройте консоль (`Ctrl + Shift + I`) и проверьте:

```javascript
// Проверка плагинов
window.Savecord.manager.getAllPlugins()

// Проверка настроек
window.Savecord.Storage.getAll()

// Проверка темы
window.Savecord.ThemeManager.getThemes()
```

## Частые ошибки

### Ошибка 1: "Module not found"

**Причина:** Не собран проект

**Решение:**
```bash
npm run build
```

### Ошибка 2: "Discord won't start"

**Причина:** Неправильный инжект

**Решение:**
```bash
# Удалите app папку
rm -rf %LocalAppData%/Discord/app-*/resources/app

# Переинжектите
node dist/injector.js
```

### Ошибка 3: "VirusTotal API limit"

**Причина:** Превышен лимит 500 запросов/день

**Решение:** Подождите до следующего дня или используйте свой API key
